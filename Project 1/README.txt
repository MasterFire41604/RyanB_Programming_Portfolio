----------------
* Instructions *
----------------
* Press left and right arrow keys to move, space to jump
* Hitting an enemy kills you, but if you jump on top of it, you get boosted up.
* Try and climb as high as you can without falling
* There are rocket boosts that will give you a big boost


--------------------------------
* Installations needed to Play *
--------------------------------
* You need python (My version is 3.10.0)
* You need to intall pygame in the terminal
	- open the command prompt and type in this command: pip install pygame
