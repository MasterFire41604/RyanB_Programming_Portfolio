----------------
* Instructions *
----------------
* Use WASD or Arrow keys to move
* Press Space to shoot
* Kill all the zombies while trying not to get killed


--------------------------------
* Installations needed to Play *
--------------------------------
* You need python (My version is 3.10.0)
	PIP should automatically be installed with this version and above
* You need to intall pygame in the terminal
	- open the command prompt and type in this command: pip install pygame
* You need to intall pytmx
	- open the command prompt and type in this command: pip install pytmx
* You need to install pytweening
	- open the command prompt and type in this command: pip install pytweening
