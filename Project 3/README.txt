----------------
* Instructions *
----------------
* Press the Up Arrow to jump
* Fly through the pipe openings and avoid touching the pipes
* Get as high of a score as you can

--------------------------------
* Installations needed to Play *
--------------------------------
* You need python (My version is 3.10.0)
	PIP should automatically be installed with this version and above
* You need to intall pygame in the terminal
	- open the command prompt and type in this command: pip install pygame
